about

Baton is a package manager built for use with the eling programming language.

It is designed to be simple and easy to use.

It is also designed to be fast and efficient.

It is also designed to be secure and reliable.

Baton is customisable and works with any server with Nodejs

Baton by default has no account system and will need to be implemented manually

To add user authentication please look for the Baton documentation

For more information please visit the Baton documentation on the Eling github
